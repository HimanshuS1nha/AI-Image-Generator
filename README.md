## Get started

1. Setup environment variables according to .env.example file

2. Install packages and run the server
- Server

   ```bash
   npm install

   npm run dev
   ```

- App

   ```bash
   npm install

   npm run start
   ```

- Set aspect ratios
- Save generated images on your device
